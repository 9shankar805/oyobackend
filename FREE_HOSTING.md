# FREE Backend Hosting Options

## 1. Render.com (BEST - Always Free)
1. Push code to GitHub
2. Go to render.com → New Web Service
3. Connect GitHub repo
4. Settings:
   - Build: `npm install`
   - Start: `npm start`
   - Plan: Free
5. Deploy automatically

## 2. Vercel (Serverless)
```bash
npm i -g vercel
cd backend
vercel --prod
```

## 3. Cyclic.sh (Simple)
1. Go to cyclic.sh
2. Connect GitHub
3. Auto-deploy

## 4. Glitch.com (Instant)
